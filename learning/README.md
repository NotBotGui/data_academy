# Learning

Self-contained learning demos, experiments, and exercises for the BI and data group.

Each piece of work gets its own numbered subfolder, e.g. `01_sales-analysis/`.
Include a `README.md` describing the objective plus any relevant `.sql`, `.ipynb`,
or `.py` files alongside it.

```
learning/
└── 01_sales-analysis/
    ├── README.md
    ├── explore.ipynb
    └── queries.sql
```
